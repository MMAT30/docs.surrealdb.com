# SurrealQL via Surrealist 

In this section, you will explore writing SurrealQL queries using [Surrealist](https://surrealist.app/query), the official query editor for SurrealDB. Surrealist is a powerful tool that allows you to write, execute, and visualize SurrealQL queries in real-time.

The Surrealist query editor provides a rich set of features, including syntax highlighting, query validation, and query execution. You can also view the query results in a tabular format, making it easy to analyze and visualize the data.

## Getting Started

To get started with Surrealist, visit the [Surrealist Query Editor](https://surrealist.app/query) and start writing your SurrealQL queries. You can use the query editor to write queries, execute them, and view the results in real-time.

## Features

Surrealist provides a number of features to help you write and execute SurrealQL queries more efficiently. Some of the key features include:

- **Syntax Highlighting**: Surrealist provides syntax highlighting for SurrealQL queries, making it easier to read and write complex queries.
- **Query view**: Surrealist provides a query view that displays the query results in a tabular format, making it easy to analyze and visualize the data.
- **Design view**: Surrealist provides a design view that allows you to visualize the query execution plan and optimize the query performance.
- **Manage Access**: Surrealist allows you to control account access to the database. This includes system users (root, namespace, and database) as well as record access.
